## 기능

1. 이름으로 로그인이 가능하다.
2. 각 이름별로 TODO 목록을 만들 수 있다.
3. 5개의 배경화면이 랜덤으로 나타난다.
4. 5개의 명언이 랜덤으로 나타난다.
5. 현재 위치와 날씨를 볼 수 있다.
